I'm building **[Walyverse](https://walyverse.fr)**, a French Minecraft universe, and I make nearly every piece of it myself:

- **The gameplay**: an ecosystem of 35+ custom Java plugins (economy, jobs, items, chat, teleportation, votes, cross-server sync, i18n, and more)
- **The web**: the network's site and store, running on a hand-made Azuriom theme (PHP, Sass, JS)
- **The infrastructure**: [Ferry](https://ferry.walyverse.fr), my own deployment platform that ships files across the whole fleet of servers, safely
- **The look**: custom 3D cosmetics, models and resource packs
